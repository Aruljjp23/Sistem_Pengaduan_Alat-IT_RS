## 📋 What is LongShuGameDev?

LongShuGameDev is a tool for people who build games. It provides an intelligent assistant that understands game development tasks. You can use it to write code, solve logic problems, and draft scripts for your projects. It works as a bridge between your ideas and your game engine.

This software uses a large language model. This means it contains a database of knowledge about how to create games. It focuses on popular engines like Unity and Unreal Engine. You can ask it questions about how to perform specific tasks. It gives you answers that you can copy and paste into your workspace.

## 🧠 Using the Assistant

When the application opens, you see a chat box. Type your question into this box and press Enter. The assistant processes your request and provides a response.

If you are working in Unity, ask questions about C# scripts. For example, you might ask how to move a character when a player presses a key. The assistant generates a script snippet for you. Read the code carefully, then copy it into your game engine.

If you use Unreal Engine, ask questions about Blueprints or C++. The assistant explains the logic you need to connect your nodes. It breaks complex tasks into small steps.

## 💡 Best Practices for Success

To get the best results, write clear questions. Include the name of your game engine. Mention the language you use.

Instead of writing "How do I make a player jump?", write "I am using Unity and C#. Show me how to make a player jump using a Rigidbody component." This level of detail helps the assistant provide accurate code every time.

If you get a slow response, wait for the cursor to stop moving. The assistant is currently processing your request. Do not close the window until the text appears in full.
